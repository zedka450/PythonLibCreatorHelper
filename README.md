#PythonLibCreatorHelper / PythonLibUpdaterHelper
A project for fastly and simply create and update Python Libs and publish her.

#---Install & use---------

download project in zip file, dezip the .zip and run this command in Terminal:
1. `cd C:\Path\To\The\Project\src`

For the Creator:
`python PLCH.py`

For the Updater:
`python PLUH.py`
